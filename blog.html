Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM (Document Object Model) using JavaScript. The most common methods are:
1.getElementById(): Selects an element by its ID.
2.getElementsByClassName(): Selects all elements with a specific class name.
3.getElementsByTagName(): Selects all elements of a specific HTML tag.
4.querySelector(): Selects the first element that matches a CSS selector.
5.querySelectorAll(): Selects all elements that match a CSS selector.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where you assign a single event listener to a parent element instead of adding event listeners to each individual child element. This is particularly useful when you have dynamically added elements or a large number of child elements.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept in the DOM (Document Object Model) where an event that is triggered on a child element "bubbles up" to its parent elements. This means that when an event occurs on a child element, it is first handled by the child element, and then the event is passed upwards through each ancestor element (parent, grandparent, etc.).

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you can create, add, and remove elements from the DOM (Document Object Model) in the following ways:
1. Creating Elements:To create a new element, you use document.createElement().
2. Adding Elements to the DOM:Once an element is created, you can add it to the DOM using methods like appendChild(), insertBefore(), or append().
3. Removing Elements from the DOM:To remove an element, you can use the remove() method or removeChild().
